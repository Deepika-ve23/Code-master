## Code-master
CodeMaster is a front-end web application designed as a coding learning and practice platform. It includes multiple pages such as Home, Features, Learning, About, Contact, Dashboard, Leaderboard, and User Profile. The website provides an interactive interface where users can explore coding resources, practice questions, track their progress, and manage their profiles.

The project is built using HTML and CSS, with a structured layout, navigation bar, and responsive design elements. It also includes sections for login/signup, settings, and a question arena, making it similar to a beginner-friendly coding platform.
